Metadata and Tagging

Metadata allows you to attach additional information to documents, making them easier to search and helping the AI understand the content more accurately.

Managing Metadata

Create a File Tag

Go to Manage File Tags → Create New File Tag. Enter a tag name (e.g. Invoice) and add fields like Invoice Number and Invoice Date.

Example Metadata Fields

For an invoice document: Invoice Number, Invoice Date, Supplier, and Total Amount.

Edit Document Metadata

Navigate to the document, right-click, select View/Edit Tag, click Edit, select the File Tag, fill in values, and click Save Changes.

Why Metadata Matters

Metadata improves search precision and helps the AI return more accurate, context-aware results when you query your documents.

Tips for Tagging

  • Create file tags before uploading documents for best results
  • Use consistent field names across similar document types
  • Metadata fields can be applied to multiple documents
  • Tags help the AI extract structured data more accurately
