Troubleshooting Common Issues

Find quick solutions to the most common problems users encounter on Doc-Scribe.ai.

Common Issues

Cannot Upload Document

Check that the file size is within limits and your internet connection is stable before retrying the upload.

AI Search Not Working

Ensure the document is selected and has finished processing. The AI cannot query a document that is still being indexed.

Cannot Access Shared Document

Check that the document was shared with you and that your permission level allows access. Contact the document owner if unsure.

Metadata Not Applying

Ensure the correct File Tag is selected for the document and all required fields are filled in before saving changes.

Quick Checklist

  • Is your file within the size limit?
  • Has the document finished processing after upload?
  • Do you have the correct permission level for shared documents?
  • Is your internet connection stable?
  • Have you selected the document before opening Chat AI?
