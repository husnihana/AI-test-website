Sharing Documents

Documents can be shared with other users or departments with controlled permission levels.

How to Share a Document

Step 1 — Open & Share

Open the document you want to share, then click the Share button.

Step 2 — Select a User

Select a user from the same department or from another department to share the document with.

Step 3 — Set Permissions

Can View: User can view and interact with the document.
Can Download: User can view and download the document.

Where It Appears

The shared document will appear in the recipient's Shared With Me section for easy access.

Permission Levels

  • Can View — interact with the document but not download
  • Can Download — view and download the document
  • Shared documents appear in recipient's Shared With Me
  • You can share across departments
