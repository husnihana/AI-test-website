User Access Control

Doc-Scribe.ai provides role-based access control to ensure users only see and interact with documents they are authorized to access.

Access Levels

Global Access

Users with Global Access can view and interact with documents across all departments and categories on the platform.

Manage Categories

Users with Manage Categories permission can create, edit, and organize departmental categories and their documents.

Document Permissions

When sharing documents, you control whether recipients can View only or also Download the document.

Private by Default

Documents in My Box are private by default. Access is only granted when explicitly shared or moved to a category.

Access Control Best Practices

  • Assign Global Access only to administrators
  • Use Manage Categories for department leads
  • Review shared document permissions regularly
  • Use My Box for drafts before moving to shared categories
  • Contact your admin if you cannot access a document you need
